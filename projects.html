ECORIDE – Carpooling Platform

ECORIDE is a full-featured carpooling platform where users can search, book, or offer eco-friendly rides. It supports role-based access for passengers, drivers, employees, and administrators, and uses Google Maps API for real-time routes and geolocation.

Includes email authentication, automated email templates (for confirmations, notifications, etc.), and a solid MySQL database structure to manage users, vehicles, preferences, and booking history. Drivers can upload profile pictures, receive real-time requests, and manage trips dynamically.

Stack: PHP, JavaScript, MySQL, Google Maps API, SMTP, CSS, HTML
